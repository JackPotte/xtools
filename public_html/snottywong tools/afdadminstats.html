Admin AfD Counter Tool

This tool will analyze the AfD closure history of a user and report statistical results. It is currently in beta and may not be reliable or accurate in all cases, and currently only works for the English Wikipedia. The tool searches through the wikitext of AfD pages which have been edited by the Username specified below, and determines how (and if) the user closed that AfD by searching for their signature in key locations. If the signature of the user doesn't always include their username, then you may need to specify an alternate username so that the tool can identify those votes (e.g. if the user's signature links to a different user page which redirects to their user page). This is not common and can be left blank in most cases.

Since computer programs are generally not good at evaluating humanity's intentions through their words, this tool is limited to searching for bolded closures and votes in AfD's. If the user you are searching for doesn't routinely bold their votes, or doesn't use certain keywords in their bolded closures/votes (e.g. Keep, Delete, etc.) then this tool will likely not work well.

You can also specify the maximum number of AfD's to search through (maximum is 250 at this time). In some cases, the number of AfD's displayed will be less than the maximum number specified. This can happen when a user edits an AfD page but doesn't vote (e.g. just leaves a comment).
