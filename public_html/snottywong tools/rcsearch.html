This tool allows for arbitrary searches through the recent changes table. All search parameters are optional, and you can use as many parameters as you like. Any parameters left blank will not be used.

Hints:
  • When using wildcards in a page title, you cannot use a wildcard in the namespace portion of the title.
  • When specifying timestamps, you can use incomplete timestamps (e.g. 20120506, 201209, 201212251237). A minimum of 4 digits must be used.
  • A maximum of 500 edits can be shown at one time.
  • If the search is taking a long time, it might mean that there are few or no edits that match your parameters.
