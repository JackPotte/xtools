Snottywong's Tools

I'm Snottywong, and I usually contribute only to the English Wikipedia under User:Snottywong. I have recently created a toolserver account and have a few tools up and running, as described below. For the moment, the tools focus on statistics relating to both deletion and new page patrolling on the English Wikipedia. If you have any questions, suggestions, or bug reports, please leave me a message on my talk page.

Tools

  • AfD Vote Counter - Analyzes statistics regarding a user's voting patterns at Articles for Deletion.
  • Admin AfD Counter - Analyzes statistics regarding an admin's AfD closure patterns.
  • New Page Patrol Report - A report on which editors have been recently patrolling new pages, their patrolling speed, and which side of the queue they're patrolling from.
  • New Page Patrol Graph - A graph of how many days worth of articles are in the queue at Special:NewPages, up to a 30-day history updated hourly.
  • User Contribution Search - Search through a page's history and return edits made by a particular user on that page.
  • User Edit Summary Search - Search through a user's edit summaries by keyword and return edits with a partial match.
